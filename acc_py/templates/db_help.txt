Interactive DB management via SQLAlchemy.

Available functions:
    e   =   edit
    d   =   delete
    w   =   write
    wl  =   write_list
    wdf =   write_list but returns dataframe
    wc  =   write_conversion
    rc  =   read_conversion
    r   =   read
    el  =   edit_list
    pc  = pretty prints categories
    
For further help, you can do: 
    - h(f)  -> prints documentation of f
    - h()   -> prints this message
